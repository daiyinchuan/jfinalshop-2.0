##  jfinalshop-2.0 项目说明

 免费开源JFinalShop让你轻松打造自己的独立商城，项目是基于JFinal开发，方便二次开发，让你快速搭建个性化独立商城，利用已有的代码减少您项目的开发周期，为您节约更多时间，去陪恋人、家人和朋友。


1、后端

* 核心框架：JFinal 2.0
* 安全框架：Apache Shiro 1.2
* 数据库连接池：Alibaba Druid 1.0
* 缓存框架：Ehcache 2.6
* 日志管理：SLF4J 1.7、Log4j2 2.7
* 工具类：Apache Commons、FastJson

2、前端

* JS框架：jquery-2.2.3
* 表格插件：Bootstrap Table
* 表单验证插件：bootstrapValidator
* 日期选择插件：Datepicker for Bootstrap
* 数据图表：echarts
* 模模：Beetl-2.2.3

2.0功能列表
---------------------------------------------------------------------------------------

 **商品管理** ，商品列表、添加商品、分类列表、添加分类、类型列表、属性列表、品牌列表。
 **订单管理** ，订单列表、收款单、退款单、发货单、退货单、会员注册协议。
 **内容管理** ，内容管理、文章管理、文章分类、导航管理、友情链接。
 **模板管理** ，动态模板管理、静态模板管理、邮件模板管理、一键网站更新、文章更新、网页底部信息。
 **管理员** ，角色管理、资源管理、收件箱、发件箱、查看日志、日志配置、商品更新。
 **网站设置** ，系统设置、配送方式、地区管理、物流公司。
